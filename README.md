# MicroService 
### Quick start
1. Create a directory for this project
```
$ mkdir microservice
$ cd microservice
$ git clone git@github.com:ChickenBenny/Social-network-web-app.git
```
2. Run the serviceby docker-compose up
```
$ docker-compose up
```
3. Architecture of this project
```
.
├─nginx
├─app_data
│   ├─app
│   └─frontend 
├─app1
├─app2
└─app3
```
